# Admin Portal

This directory is designated for the **Drip Doggy** Administration and Management dashboard application.

## Role & Responsibilities
The admin application handles backend configuration interfaces, inventory management, user auditing, order fullfillment metrics, and banner setup tasks.

## Planned Stack
- **Framework**: React / Next.js
- **Routing & State**: React Router / Redux Toolkit
- **Styling**: Tailwind CSS
- **Integrations**: Spring Boot API integration for inventory updates
